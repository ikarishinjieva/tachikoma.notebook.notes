# 有用的结论

```
该研究的主要结论如下，并附带例子说明：

**1. 领域特定词汇和预训练对于创建稳健的模型至关重要。**

* **结论:** 使用领域特定数据（PubMed摘要）从头预训练的模型（PubMedBERT-LARGE）比使用通用领域数据预训练后在生物医学数据上继续预训练的模型（BioBERT-LARGE, BlueBERT-LARGE）表现更佳，尤其在面对小数据集和复杂任务时，稳定性更好，性能更高。
* **举例:** 在BLURB基准测试中，PubMedBERT-LARGE在问答任务（BioASQ和PubMedQA）上的表现优于BioBERT-LARGE和BlueBERT-LARGE，尤其是在BioASQ这种小数据集的任务上，差距更加明显。

**5.  简单的优化调整（例如更长的训练时间和ADAM偏差修正）是必要的，但不足以完全解决微调不稳定性问题。**

* **结论:**  这些调整可以提升性能和稳定性，但对于解决生物医学领域小数据集带来的挑战仍然不够。
* **举例:** 在BIOSSES任务上，即使使用了更长的训练时间和ADAM偏差修正，使用单序列预训练的模型性能仍然很低，说明需要更高级的稳定性方法。
 
**6. 分层自适应方法（冻结底层、分层学习率衰减和重新初始化顶层）可以有效提高微调稳定性，但最佳策略取决于任务和预训练设置。**

* **结论:**  冻结底层适用于标准BERT-BASE模型；分层学习率衰减适用于BERT-LARGE和ELECTRA模型；重新初始化顶层适用于低资源文本相似度任务。
* **举例:** 在BIOSSES任务上，重新初始化顶层对所有模型都有显著的提升；在BioASQ任务上，BERT-BASE模型受益于冻结底层，而ELECTRA模型受益于分层学习率衰减。

 
**7.  大模型的微调稳定性可以通过分层自适应方法和优化调整得到改善。**

* **结论:**  结合这些技术，可以使大模型在生物医学NLP任务上达到更高的性能。
* **举例:** 在BioASQ任务上，使用分层学习率衰减的PubMedELECTRA-LARGE模型达到了最佳性能。

总而言之，该研究系统地探索了生物医学NLP中大语言模型微调的稳定性问题，并提出了一系列有效的改进策略。这些结论对于指导生物医学NLP领域的模型微调实践具有重要的参考价值。
```
