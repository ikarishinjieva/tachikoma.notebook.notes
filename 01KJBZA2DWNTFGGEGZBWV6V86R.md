# 主要思路

  - 划定LLM的知识边界: 将训练问题 在有/无RAG的情况下, 进行答案效果对比, 确定哪些问题是LLM的自有知识
  - 将知识边界形成可使用的模型, 可以选择以下四种方法其一:
    - 直接提问: 直接询问 LLM 是否需要额外信息来回答问题。

    - 上下文学习: 通过一些已知问题和对应答案的示例，引导 LLM 判断是否需要额外信息。

    - 训练分类器: 使用已知问题和标签训练一个分类器，用于预测新问题是否需要检索信息。

    - 最近邻搜索: 使用预训练的编码器计算新问题与已知问题的语义相似度，通过 k 近邻搜索找到最相似的已知问题，并根据其标签预测新问题是否需要检索信息。

  - 使用知识边界模型, 判断新的问题是否用LLM的自有知识能解答, 从而判断是否需要召回文档
