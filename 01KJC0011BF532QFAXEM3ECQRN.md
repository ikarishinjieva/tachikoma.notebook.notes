# 主要内容

讨论了 使用 大批次 和 使用 小批次 数据, 对模型进行参数更新, 会导致模型的泛化性的变化

结论: 使用大批次数据时, 模型的泛化性变差

核心原因:

  - 使用大批次数据, 是削减了批次内的数据的"噪声", 当模型训练陷入次优解时, 不容易从次优解的吸引盆中走出来
  - 使用小批次数据, 噪声随机性比较大, 导致从次优解的吸引盆中逃逸出来的可能性变高, 直到找到一个更大的盆地, 也就找到了更有泛化性的解

其他信息: 

  - 论文中, 认为次优解是一个陡峭的盆地, 而更有解是一个平摊的盆地, 更有泛化性
