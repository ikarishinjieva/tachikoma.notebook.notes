# 结论

  1. 在基线测试中, 两步微调最终任务, 对于长度6的识别准确率在25%左右. 提高epoch可以提高长度4/5的识别准确率, 但不能提高长度6的准确率
  2. 增加填空任务的训练
     1. 增加长度1-5的填空任务数据训练, 可以轻微提高对长度6的识别准确率到30%
     2. 增加长度1-6的填空任务数据训练, 需要提高所有任务的epoch (4/2/2), 可以提高识别准确率到40%. 其中大部分错误case的原子数识别仍然是5
     3. 在此基础上, 继续提高epoch, 会起到副作用
     4. 使用两阶段的填空任务训练(长度1-5 + 长度1-6), 在同样的epoch下, 能提高对长度6的预测率到36%. 认为两阶段训练是有效的.
  3. 增加原子数识别任务的训练
     1. 未见明显提升, 且有所下降
     2. 切换输出为json格式 (跟最终任务的输出格式相似), 也不能提升
