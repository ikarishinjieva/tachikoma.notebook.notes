# 主要思路

  - 在提示词中提供 "方向性指引" 能增强LLM的输出质量
  - 方向性指引 包括: 
    - 1\. **关键词列表 (Keyword List)**：用于摘要任务，引导 LLM 生成包含指定关键词的摘要。

    - 2\. **对话行为 (Dialogue Act)**：用于对话响应生成任务，引导 LLM 生成符合对话流程和目标的回复。

    - 3\. **触发词 (Trigger Phrases)**：用于思维链推理任务，引导 LLM 进行逐步推理，例如使用 "让我们一步一步地思考"。

  - 值得借鉴的是:
    - "关键词列表" 能增强 提示词的效果
    - 如何训练 生成 "方向性指引" 的模型

# 如何训练 生成 "方向性指引" 的模型

训练步骤: 

  - **模型选择**: 选择一个小型可调语言模型作为策略模型，例如 T5。
  - **监督微调 (SFT)**: 使用准备好的训练数据对策略模型进行微调，使其能够根据输入文章生成关键词列表。
  - **强化学习 (RL)**:
    - 使用 SFT 训练好的策略模型初始化 RL 的策略网络。
    - 定义奖励函数：可以使用 ROUGE 指标来衡量 LLM 生成的摘要与参考摘要之间的相似度，并将 ROUGE 分数作为奖励。
    - 使用强化学习算法 (例如 PPO) 对策略网络进行训练，使其能够生成能够最大化奖励的关键词列表。

为什么要先使用SFT, 再使用RL:

  - SFT 微调后的模型, 为RL提供了一个很好的起点. 否则 只使用RL需要很大的成本才能学会 "当前任务有什么要求 (比如如何输出)". 
  - 只使用SFT, 微调的训练数据构造了有限的空间, SFT后的模型很难突破这个空间限制. 而RL通过与环境交互，根据奖励函数来学习最优策略。这使得 RL 可以探索 SFT 未覆盖的提示空间.
