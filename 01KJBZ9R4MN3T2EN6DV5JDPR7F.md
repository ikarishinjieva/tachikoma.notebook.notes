# 主要思路

  - 思维链CoT, 是使用few-shot提示LLM 用什么样的逻辑推导过程
  - CoT的问题, 如何选择最有用的问题(的回答过程), 作为few-shot
    - 主要是因为LLM上下文长度有限
  - 对CoT最有用的问题 == 不确定度最高的问题 (才需要人类进行标注)
  - 可使用的不确定度的度量:
    - 论文中提到的用于 Active-Prompt 方法的四个不确定性度量，旨在量化模型对特定问题答案的不确定程度，以便选择最不确定的问题进行人工标注，从而提高模型性能。以下是这四种度量的详细描述：

**1. 分歧度 (Disagreement):**

* **原理:** 该度量基于模型对同一问题多次预测结果的差异性。如果模型对某个问题的多个预测结果差异很大，则认为该问题的不确定性较高。  
* **计算方法:**  
1\. 使用 LLM 对每个问题进行 k 次独立预测，得到 k 个答案。  
2\. 计算 k 个答案中唯一答案的数量 h。  
3\. 分歧度 u = h / k。  
* **特点:** 实现简单直观，适用于判断模型在答案空间中的不确定性。

**2. 熵 (Entropy):**

* **原理:** 熵是信息论中的概念，用于衡量一个随机变量的不确定性。在 Active-Prompt 中，熵用于衡量模型对某个问题预测结果的概率分布的不确定性。  
* **计算方法:**  
1\. 使用 LLM 对每个问题进行 k 次独立预测，得到 k 个答案。  
2\. 计算每个唯一答案在 k 个答案中出现的频率，即 Pθ(aj|qi)。  
3\. 根据公式 u = argmaxi \- Σkj=1 Pθ(aj|qi) * ln(Pθ(aj|qi)) 计算熵。  
* **特点:** 能够更准确地反映模型预测结果的概率分布，适用于答案空间较大或答案分布不均匀的情况。

**3. 方差 (Variance):**

* **原理:** 方差用于衡量一组数据的离散程度。在 Active-Prompt 中，方差用于衡量模型对某个问题多个预测结果的数值差异。  
* **计算方法:**  
1\. 使用 LLM 对每个问题进行 k 次独立预测，得到 k 个答案。  
2\. 对 k 个答案进行标准化处理，以消除问题中出现的数字大小对结果的影响。  
3\. 根据公式 u = argmaxi Σkj=1 (aj - ̄a)2 / (k - 1) 计算方差，其中 ̄a 为 k 个答案的平均值。  
* **特点:** 适用于答案为数值型的问题，例如算术推理问题。

**4. 自信度:**

* **原理:** 该度量基于模型对自身预测结果的自信程度。如果模型对某个问题的预测结果不太自信，则认为该问题的不确定性较高。  
* **计算方法:**  
1\. 将问题和模型生成的答案重新组织成一个新的模板。  
2\. 使用 LLM 对该模板进行评估，并输出其对该答案的自信程度，例如 “非常自信”、“自信”、“不太自信”、“错误答案”。  
* **特点:** 直接利用了模型自身的评估机制，但容易受到 LLM 过度自信问题的影响，效果可能不如其他三种度量方法稳定。

论文作者通过实验发现，分歧度、熵和方差三种度量方法在 Active-Prompt 中表现良好，而自Confidence 方法效果不佳。因此，在实际应用中，建议优先考虑使用分歧度和熵作为不确定性度量。

    - 度量的效果差异:
    - 论文中提到了对四种不确定性度量的效果进行了实验比较，但并没有进行非常详细的分析和对比。根据论文中的描述以及对这些度量方法的理解，可以总结出以下效果差异：

**表现较好的度量：**

* **分歧度 (Disagreement):** 在实验中，分歧度表现出与其他方法相当或更优的效果。这可能是因为它直接反映了模型在不同预测结果之间的差异，能够有效识别出模型难以确定答案的问题。  
* **熵 (Entropy):** 熵也表现出与分歧度相当的良好效果。相较于分歧度，熵能够更准确地反映模型预测结果的概率分布，因此在答案空间较大或答案分布不均匀的情况下可能更具优势。

**表现稍逊的度量：**

* **方差 (Variance):** 方差主要适用于答案为数值型的问题，例如算术推理问题。在其他类型的推理问题上，方差的效果可能不如分歧度和熵。

**效果不佳的度量：**

* **自Confidence:** 实验结果表明，自Confidence 方法的效果不佳。这可能是因为大型语言模型 (LLM) 容易出现过度自信的问题，即使在预测错误的情况下也可能给出较高的置信度，导致该度量方法的可靠性较低。

**总结:**

* 分歧度和熵是较为通用且效果较好的不确定性度量方法，适用于各种类型的推理问题。  
* 方差适用于答案为数值型的问题，但在其他类型的推理问题上效果可能不如分歧度和熵。  
* 自Confidence 方法容易受到 LLM 过度自信问题的影响，效果不佳，不建议使用。

需要注意的是，以上只是一些基于论文描述和对度量方法理解的推测，具体的效果差异还需要更详细的实验结果和分析才能得出更准确的结论。
