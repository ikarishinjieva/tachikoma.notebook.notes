## 摘要
记录 SQLShift 官网介绍框架，涵盖 Oracle 到 OceanBase 的数据库迁移特性差异。重点描述复杂对象自动化迁移技术（动态 SQL 拼接、游标循环）及数据一致性保障中的精度差异处理。

## 关键概念
- 数据库方言特性：不同数据库系统之间的语法和功能差异
- 复杂对象自动化迁移：将存储过程、函数等复杂数据库对象自动转换到目标数据库
- 动态 SQL 拼接：在存储过程中动态构建 SQL 语句的技术
- 游标循环：用于遍历查询结果集的数据库编程技术
- 数据一致性保障：确保源数据库和目标数据库数据精度和一致性
