## 摘要
记录 ChatGLM-6B 模型在 SQL 规则验证任务上的微调实验，通过将 SQL 结构分析过程精细化写入训练数据，尝试让结构分析影响后续规则判断流程。测试显示模型已能正确分析 SQL 嵌套结构，但对 limit 子句的数值感知和结论生成仍有问题，需要更多训练数据增强。

## 关键概念
- 结构分析关联：将 SQL 结构分析步骤显式写入训练数据，使其影响后续规则验证的推理过程
- 训练数据复制：将少量测试样本复制多份以满足模型训练的最小数据量要求
- 规则验证流程：先分析 SQL 结构（主语句 + 子查询），再对每个语句逐一进行规则检查，最后综合得出结论
