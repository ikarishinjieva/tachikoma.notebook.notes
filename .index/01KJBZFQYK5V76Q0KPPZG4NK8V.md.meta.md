## 摘要
记录通过 Docker 容器连接 Superset 后端 PostgreSQL 数据库的操作命令。使用 docker exec 直接在 superset_db 容器中执行 psql 命令访问 examples 数据库。

## 关键概念
- Docker exec: 在运行中的 Docker 容器内执行命令的工具
- psql: PostgreSQL 数据库的命令行交互工具
- Superset: Apache 开源的数据可视化和 BI 平台
