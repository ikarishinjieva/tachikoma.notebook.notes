## 摘要
记录在 Windows 环境下使用 pywinauto 库进行 RPA 自动化调试的过程。通过控制记事本应用程序验证了 pywinauto 的窗口操作、菜单选择和控件交互功能。

## 关键概念
- pywinauto: Python Windows 自动化库，用于控制 Windows GUI 应用程序
- Application: pywinauto 的核心类，用于启动和管理应用程序进程
- menu_select: 通过菜单路径选择执行菜单命令的方法
- type_keys: 模拟键盘输入到控件的方法
