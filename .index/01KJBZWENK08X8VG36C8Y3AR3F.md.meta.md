## 摘要
GSPO 将训练过程中的 token 级别修正改为序列级别差异以增强效果。GRPO 通过计算同一查询的多个响应组内相对优势来避免价值模型需求，但其 token 级重要性权重设计存在根本问题。

## 关键概念
- GSPO: 序列级策略优化，将 token 级修正改为序列级差异
- GRPO: 组内相对优势策略优化，通过组内 Z-Score 计算优势
- 价值模型: 传统 RL 中用于评估状态/动作优势的模型，GRPO 试图避免其需求
- 重要性采样: 用旧策略采样样本来估计新策略期望的技术
- Token 级权重: GRPO 对每个 token 单独计算重要性权重，存在高方差噪声累积问题
