## 摘要
笔记介绍了 DSPy 及其他 prompt 优化项目（gpt-prompt-engineer、AutoPrompt、SAMMO）的核心逻辑与特点对比。指出 DSPy 在流行度和文档完善度上优于其他项目。

## 关键概念
- gpt-prompt-engineer: 通过 LLM 生成 prompt 并进行两两对战，用 ELO 分排名评估效果
- AutoPrompt: 用户输入任务与初始 prompt，通过生成测试样例、结果分析、循环优化提升
- SAMMO: 微软推出的 prompt 优化项目，支持重写、精简、删除、格式转换等多种修改方式
- DSPy: 斯坦福开发的 prompt 优化框架，流行度和文档完善程度较高
