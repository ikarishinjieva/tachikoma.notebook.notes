## 摘要
记录压敏薄膜传感器通过压感模块接入 Arduino 的硬件连接方法和软件配置流程，包含 Arduino 代码和 Python 串口读取脚本。实测最大压力值 663，验证了传感器数据读取功能正常。

## 关键概念
- 压敏传感器：将压力变化转换为模拟电压信号的薄膜传感器
- Arduino A0：模拟输入引脚，读取 0-1023 的模拟值
- 串口通信：Arduino 通过 USB 串口向 Python 程序发送传感器数据
- 波特率 9600：Arduino 与 Python 通信的串口传输速率
