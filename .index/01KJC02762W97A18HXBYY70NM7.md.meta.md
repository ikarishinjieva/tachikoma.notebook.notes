## 摘要
记录在阿里云无影云电脑上使用 Docker 版 Isaac ROS 进行机械手教程的完整配置流程。涵盖 nvidia-container-toolkit 安装、ROS2 工作区配置、容器内依赖安装与编译步骤。

## 关键概念
- nvidia-container-toolkit: 使 Docker 容器能够正确使用 NVIDIA GPU 的运行时工具
- ISAAC_ROS_WS: Isaac ROS 开发工作区的环境变量，指向工作区根目录
- run_dev.sh: Isaac ROS 提供的脚本，用于启动开发环境 Docker 容器
- colcon build: ROS2 的工作区编译工具，用于构建 ROS 包
- rosdep: ROS 依赖管理工具，用于安装系统级依赖包
