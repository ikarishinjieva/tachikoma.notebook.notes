## 摘要
记录 SQL 优化模型微调的数据构建过程梳理与重构，分析训练数据生成流程中的问题。通过两轮训练实验发现改写分析与改写指令不匹配等关键问题，提出简化训练数据生成方案。

## 关键概念
- 两阶段验证: 阶段 1 进行 SQL 优化分析，阶段 2 根据分析流程改写 SQL
- 改写指令: 对 SQL 结构分析后生成的具体修改指导，需与改写分析一一匹配
- 聚类中心 SQL: 从 SQL 数据集中抽取的代表性 SQL，作为数据构建的基础
- 定向增强数据: 在聚类中心 SQL 中增加指定方向的错误片段生成的训练数据
