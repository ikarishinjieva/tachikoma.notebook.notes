## 标签
opencode, AI 编程助手，配置教程，API 代理，会话管理，服务器部署

## 摘要
记录 opencode AI 编程助手的完整配置流程，包括 API 代理设置、服务器 SSH 连接、配置文件创建和认证信息。涵盖模型切换、会话导出与恢复等核心功能使用方法。

## 关键概念
- opencode: AI 编程助手工具，支持通过命令行切换模型和管理会话
- API 代理：通过 yansd666.com 代理访问 Anthropic API 服务
- 会话管理：使用/export 导出会话，通过 session list 和-s 参数恢复对话
- 配置文件：opencode.json 配置提供商 baseURL，auth.json 存储 API 密钥
