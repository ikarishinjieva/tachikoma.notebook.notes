## 摘要
记录 ChatGLM 微调用于 SQL 规则审核的第二次迭代，核心是重构数据生成器——只输入最终结果并从中分离各步骤训练数据。笔记包含完整的模型推理示例，展示如何用 AST 分析判断 SQL 是否符合"select 语句不能使用 order by"规则。

## 关键概念
- 数据生成器重构: 只输入最终结果，从中分离出每个步骤的训练数据
- AST(抽象语法树): 将 SQL 解析为结构化表示，用于规则检查
- SQL 规则审核: 使用微调后的模型判断 SQL 是否符合预定义规则
- 子查询分析: 区分直接使用和子查询中的使用，进行分层检查
