## 标签
Oracle, 数据库维护, RMAN, 日志清理, SQL 查询, 系统管理

## 摘要
记录 Oracle 数据库维护技巧，包括使用 RMAN 清理归档日志、重启数据库的操作流程。提供查询用户表数量、对象数量和表 DDL 定义的 SQL 命令示例。

## 关键概念
- RMAN: Oracle 备份恢复管理器，用于删除归档日志等维护操作
- sysdba: Oracle 数据库管理员权限，用于执行关机和启动操作
- all_tables: 系统视图，用于查询数据库中所有表的信息
- all_objects: 系统视图，用于查询数据库中所有对象的信息
- dbms_metadata: Oracle 包，用于获取数据库对象的 DDL 定义
