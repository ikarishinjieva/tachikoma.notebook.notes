## 摘要
记录使用 AI 工具分析 TikTok 视频内容的尝试，以及 Google Veo API 生成视频的实践过程。分析 MoneyPrinterTurbo 项目的文本生成、视频处理和字幕生成流程，提出分镜图片先生成再拼接的视频生成算法思路。

## 关键概念
- Google Veo API: Google 提供的文生视频和图生视频生成 API
- MoneyPrinterTurbo: 自动化视频生成项目，整合文本、音频、视频素材
- 分镜生成算法: 先生成分镜点图片，再用首尾图片生成视频片段后拼接
- 对口型功能: 即梦等工具提供的音频驱动口型同步功能
