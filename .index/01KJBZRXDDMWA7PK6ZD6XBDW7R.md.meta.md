## 标签
SQL 优化，模型微调，聚类算法，验证集，Text2SQL，两阶段聚类

## 摘要
笔记记录了为 SQL 优化模型微调构建验证集的方案，包括从多个公开 Text2SQL 数据集中筛选合适的校验集，以及使用两阶段聚类（Tf-idf 粗聚类 + 编辑距离精聚类）从 SQL 空间抽取代表性样本的方法。同时探索了通过规则改写生成"错误 SQL"的技术，用于构建模型训练的对抗样本。

## 关键概念
- 两阶段聚类: 先用 Tf-idf 降维粗聚类，再用编辑距离精确聚类，平衡速度与精度
- 编辑距离度量: 用带权编辑距离衡量 SQL 关键字序列的相似性，但需构建完整距离矩阵
- 投影下推规则: SQL 优化规则，通过删除子查询中无意义的列来提升性能
- 对抗样本生成: 基于规则将正常 SQL 改写为违反优化规则的"错误 SQL"
