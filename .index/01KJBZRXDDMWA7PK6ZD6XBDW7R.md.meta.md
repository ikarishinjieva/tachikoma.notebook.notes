## 摘要
本文记录了为 SQL 优化模型微调寻找合适校验集的方案，包括多个 Text2SQL 数据集的调研。核心采用两阶段聚类算法（TF-IDF 初步聚类 + 编辑距离精确聚类）对 SQL 进行采样，并探索了根据优化规则构造"违规"SQL 的方法用于模型训练评估。

## 关键概念
- 两阶段聚类：先用 TF-IDF 降维初步聚类，再用编辑距离构建距离矩阵进行精确聚类
- 投影下推：SQL 优化规则，通过删除子查询中无意义的列来减少 IO 和网络开销
- 编辑距离：用于衡量 SQL 关键字序列相似性的距离度量
- Text2SQL：将自然语言转换为 SQL 查询的任务，用于构建验证数据集
- LSH（局部敏感哈希）：一种通过阈值划分类的近似最近邻搜索算法
