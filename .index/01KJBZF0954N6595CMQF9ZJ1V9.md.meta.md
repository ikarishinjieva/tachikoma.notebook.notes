## 摘要
使用 google.generativeai 库批量删除 Gemini 用户上传文件的 Python 代码。通过 genai.list_files() 遍历文件并调用 delete() 方法清理。

## 关键概念
- genai.configure: 配置 Gemini API 密钥和传输协议
- genai.list_files: 列出用户上传到 Gemini 的所有文件
- f.delete: 删除单个文件对象的方法
