## 摘要
记录了一个完整的带货视频生成工作流：从图片道具替换、分段视频生成到最终拼接配音。关键技术点包括控制视频末帧姿态避免虚影、使用云端 API 解决 IO 瓶颈。

## 关键概念
- Flux-pro/kontext/max: 用于替换图片中人物手持道具的图像生成模型
- 末帧静止控制: 让视频最后一秒人物直视镜头并保持静止，避免分段拼接时产生虚影
- 云端视频拼接: 使用阿里云 API 进行视频拼接，避免本地 IO 资源不足
