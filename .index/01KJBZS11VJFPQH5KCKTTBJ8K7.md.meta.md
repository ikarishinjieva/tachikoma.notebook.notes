## 标签
SQL 优化，模型训练，渐进式掩码，反事实增强，结构分析，微调

## 摘要
探索 SQL 优化模型训练的多种方法，包括对抗模型、渐进式掩码训练和反事实增强训练。通过 DeepSeek 生成 SQL 结构分析数据，并设计填空任务验证模型对 SQL 结构的认知能力。

## 关键概念
- 渐进式掩码训练: 分阶段掩码不同部分（结构分析/优化方案），逐步提升模型联合推理能力
- 反事实增强训练: 通过对比错误优化与正确优化样本，让模型学习语义保持的优化边界
- SQL 结构分析: 将 SQL 查询分解为层次化故事框架，明确各子查询的功能定位
