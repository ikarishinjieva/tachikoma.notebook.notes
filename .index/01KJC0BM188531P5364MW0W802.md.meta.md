## 摘要
记录在 Cursor 中配置 Playwright MCP 服务器的完整过程，包括 nvm/npm 安装和浏览器配置。测试验证了通过 MCP 控制浏览器搜索的功能，同时发现获取完整 HTML 会导致 context 超限的问题。

## 关键概念
- MCP Servers: Cursor 中用于扩展 AI 能力的协议服务器配置
- Playwright MCP: 通过 Playwright 实现浏览器自动化的 MCP 服务
- nvm: Node 版本管理工具，用于安装特定版本的 Node.js/npm
- Context 超限: MCP 获取完整 HTML 时超出 AI 上下文限制导致记忆丢失
