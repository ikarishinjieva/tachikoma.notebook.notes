## 摘要
详解 GRPO 训练的四个核心步骤：生成阶段、奖励计算、优势函数计算、策略比率裁剪。重点分析非对称裁剪的上下界设计原理，以及 token 级优势计算的信用分配问题与解决方案。

## 关键概念
- 策略比率裁剪: 通过限制新旧策略概率比的更新幅度（上界 1.28/下界 0.8）防止训练过于激进
- 优势函数 (Advantage): 衡量每个 token 选择相比基准的优劣程度，用于指导策略更新方向
- 信用分配问题: 将整体奖励合理分配到每个 token 的核心挑战
- 非对称裁剪: DAPO 改进，上界 (1.28) 允许快速学习好模式，下界 (0.8) 保守抑制避免崩溃
- GAE 优势估计: 使用时序差分误差进行 token 级优势计算的标准方法
