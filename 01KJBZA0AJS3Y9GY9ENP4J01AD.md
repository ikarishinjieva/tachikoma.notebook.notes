# 主要思路

论文主要探讨了LLM对自己的认知边界 认知是否清晰

论文提出了 先验指标和后验指标, 这两组指标都是通过大模型提示词获得

通过这两组指标的结果, 文章对大模型的认知边界做出了判断: 

在论文的实验结果部分，先验判断和后验判断是用于评估 LLM 对自身事实性知识边界感知能力的两个指标类别，每个类别下又包含更具体的指标。

**1. 先验判断 (Priori Judgement):**

* **目的**: 评估 LLM 在回答问题 **之前** 对自身能否正确回答的判断能力。  
* **具体指标**:  
* **放弃回答率 (Give-up rate):** LLM 选择不回答问题 (例如，回答“不知道”或“无法回答”) 的比例。比例越高，说明 LLM 越倾向于认为自己无法回答，也即对自身知识边界感知越保守。  
* **正确放弃率 (Right/G):** 在 LLM 放弃回答的问题中，实际能够正确回答的比例。该指标越高，说明 LLM 在放弃回答时，往往是对的，能够准确识别自身不会的问题。  
* **正确回答率 (Right/¬G):** 在 LLM 没有放弃回答的问题中，实际能够正确回答的比例。该指标越高，说明 LLM 在选择回答问题时，往往能够给出正确答案，也即对自身能够回答的问题有较好的把握。

**2. 后验判断 (Posteriori Judgement):**

* **目的**: 评估 LLM 在回答问题 **之后** 对自身答案正确性的判断能力。  
* **具体指标**:  
* **评估正确率 (Eval-Right):** LLM 将自己的答案评估为“正确”的比例。该指标越高，说明 LLM 对自身答案的信心越高。  
* **评估准确率 (Eval-Acc):** LLM 对自身答案的评估 (正确或错误) 与实际情况相符的比例。该指标越高，说明 LLM 对自身答案的判断越准确。

**实验结果分析:**

通过分析这些指标，论文发现 LLM 在感知自身事实性知识边界方面存在以下问题：

* **过度自信:** LLM 的放弃回答率通常较低，即使面对实际上无法正确回答的问题，它们也倾向于尝试给出答案。  
* **判断不准确:** LLM 的正确放弃率和评估准确率都不够高，说明它们在判断自身能否正确回答问题、以及判断答案是否正确方面都存在不足。

论文还发现，检索增强技术能够有效提高 LLM 的先验判断和后验判断能力，使其能够更准确地感知自身的事实性知识边界。
