# 主要方法
    
    
    AMA方法主要解决的问题是: 微小的提示词改动, 会造成输出的大幅变化. 所以AMA通过使用多个提示词的综合概率计算, 来抑制这种不正常的变化.

```
AMA 步骤：

**1. 准备多个提示词链**

* 每个提示词链包含一个 question() prompt 和一个 answer() prompt。 这些 prompts 可以是事先设计好的固定组合，也可以从 prompt 库中随机组合。

**2. 对输入和输出进行改写**

* 对于每一个提示词链, 对输入和输出进行改写, 相当于将LLM的数据流程变为: 输入(改) -> LLM -> 输出(改)
* 具体来说：
    * 输入改写： question() prompt 将原始输入改写成不同类型的问句，例如是非题、选择题、填空题或自由问答题。
    * 输出改写：  answer() prompt 引导模型生成不同风格的答案，并将其映射到任务的输出空间。

**3. 综合所有提示词链 (用输入改和输出改 代表 某个提示词链的能力), 生成一个概率图. 输入 -> LLM -> 输出的流程中, 输出是综合了各个提示词链的综合概率 而得出的** 
	3.1 其中: 会使用到依赖图, 来分析各个提示词链的正交性, 以调节概率图, 使得提示词链构成的概率空间尽量正交

```
